超市收银台顾客排队模拟
不考虑超市有人进入，以超市的某一收银台作为研究对象，顾客到达收银台的时间间隔服从平均时间为10s的负指数分布。
每个顾客的服务时间服从正态分布。
模拟顾客在收银台的平均逗留时间和收银系统的服务强度（服务占总时间比率）
由于存在随机性，故在固定超市初始总人数Sum_amount_person的情况下，需要多次运行仿真，得到此程序中收银强度、顾客平均逗留时间随实验次数而变化的图。
修改程序中的概率分布，可以得到不同的运行结果。
通过记录其他中间变量，可以得到顾客排队时间、结账时间、收银台收银时间、空闲时间等
