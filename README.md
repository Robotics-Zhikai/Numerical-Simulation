基于时间流模拟的数值仿真。大致思路就是将时间离散化为极小量，每个时间该做什么用程序去模拟，进而得到一些已知初值的动态系统的特性或状态

## 排队系统仿真模拟示意
初始有40人，进入收银台间隔时间服从指数分布，收银持续时间服从高斯分布，整个随机过程是维纳过程。
